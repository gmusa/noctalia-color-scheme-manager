# Noctalia Color Scheme Manager

GUI application to create and edit color themes for the [Noctalia](https://github.com/Yaulendil/noctalia) Wayland compositor.

Built with Python + GTK4 + libadwaita.

## Features

- Create and edit color themes
- Live preview of changes
- Support for Material Design tokens and terminal colors
